wot++ as a service

About

Wot++ is a language created by Jackojc for writing documents that allows you to build up your own library of abstractions or use those made by others.

The language has some interesting features like meta-programming, the ability to invoke a shell, bit-level manipulation, recursion and conditionals.

You can find out more about Wot++ on github.

Upload a wot++ file to this site (supports drag and drop/multiple uploads) to receive the compiled output.
